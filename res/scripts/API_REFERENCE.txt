
BLIND JUMP API REFRENCE 0.0.1

The Blind Jump core engine exposes a bunch of functions on the Lua
side of things, their scemantics and calling conventions are provided
here for your convenience.


ENTITY

entity.create(classname) -> handle

entity.getPosition(handle) -> x, y

entity.setPosition(handle, x, y)

entity.setKillFlag(handle)

entity.getKillFlag(handle) -> isSet

entity.emitSound(handle, soundName, minDistance, attenuation, looping)

entity.getState(handle) -> state

entity.setState(handle, state)

entity.getTimer(handle, timerno) -> timerval

entity.setTimer(handle, timerno, timerval)


SYSTEM

system.getScreenSize() -> width, height

system.getDeltaTime() -> microseconds

system.registerClass(classname)
